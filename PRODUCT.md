# Product

## Register

product

## Users

Researchers, policymakers, and development-finance analysts studying how Multilateral Development Banks (MDBs) align project portfolios with the UN Sustainable Development Goals (SDGs). They use the dashboard for exploratory analysis: comparing MDBs, years, countries, and SDG co-tagging patterns during desk research or peer review.

## Product Purpose

Provide an interactive overview of MDB operations (2016–2025) classified into 17 SDGs via NLP. Success means users can quickly grasp portfolio composition, drill into specific MDBs or countries, and extract insights without running the underlying data pipeline.

## Brand Personality

Authoritative, research-grade, transparent. Scholarly confidence without corporate sterility—the voice of a researcher sharing a useful tool with peers.

## Anti-references

- Generic SaaS dashboard clichés (hero metrics, cream backgrounds, gradient accents)
- Over-decorated data viz that competes with the SDG color system
- Consumer-style marketing polish that undermines research credibility

## Design Principles

1. Data first — charts and maps are the product; chrome stays quiet
2. Familiar analyst UX — standard nav, sidebar filters, predictable interactions
3. SDG colors are canonical — do not override UN palette semantics
4. Density is a feature — researchers expect information-rich views
5. Trust through precision — accurate labels, consistent terminology (MDB, SDG, project count vs amount)

## Accessibility & Inclusion

WCAG 2.1 AA target. Keyboard navigation for filters and controls. Respect `prefers-reduced-motion`. Sufficient contrast for body text and chart labels.
